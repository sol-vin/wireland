# wireland

An esoteric logic simulator inspired by TTL. 

## Installation

1. Install Crystal
2. `shards install`
3. Install Raylib from `lib/raylib-cr/rsrc`
4. `shards build app`

## Usage

Run `./bin/app`
Load `.png` files from `rsrc`
