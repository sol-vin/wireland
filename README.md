# wireland

An esoteric logic simulator inspired by TTL.



## Installation

1. Install Crystal
2. `shards install`
3. Install Raylib from `lib/raylib-cr/rsrc` or install raylib from source
4. `shards build app`

## Usage

Run `./bin/app` and drag and drop palette and png files into the window.
